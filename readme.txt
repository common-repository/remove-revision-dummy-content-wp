== Installation ==

1. Upload `remove-revision-dummy-content-wp` folder to your `/wp-content/plugins/` directory.
2. Activate the plugin from Admin > Plugins menu.
3. Once activated you should check with Settings > Remove Revision Dummy Content WP
4. Use Remove revision.
